# AD hosts
### 依然的爱
*替换hosts来屏蔽广告*

### Links
* [GitHub](https://github.com/E7KMbb/AD-hosts)

### 介绍
本模块会以Systemless的形式替换hosts进行广告屏蔽。

### 注意
使用时请关闭其它的hosts模块，并打开Systemless hosts选项。
